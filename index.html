COMMANDS 

Isabelle Bot is of course operated by commands! Each command begins with the prefix '$'   

  • Animalese Translator
  • Tune Maker
  • NH Lookup
  • KK Slider
  • NH Soundtrack
  • Switch Codes
  • Settings

Animalese Translator 

Isabelle Bot can translate anything you type into Animalese, the language of Animal Crossing!

This is possible thanks to HenryIsHuman!

  • $translate tomnook your sentence
  • $translate timmy your sentence
  • $translate tommy your sentence
  • $join (Joins the voice chat you're in)
  • $leave (Leaves the voice chat you're in)

Island Tune Maker

Isabelle Bot can write your island tune! Using a scale of notes from "gabcdefGABCDE-_", you can write a tune up to 16 notes and it will play in the voice chat that you're in!

  • $tune your tune

Database Commands

Isabelle Bot will search the web for information about various things in Animal Crossing: New Horizons!

  • $bug (Search bugs by name, location, and month)
  • $fish (Search fish by name, location, and month)
  • $villager (Search villagers by name, birthday, and species)
  • $recipe (Search recipes by item/furniture names)
  • $material (Search recipes by material used)

KK Slider Composer

Isabelle Bot can be used to create authentic KK Slider music using the power of MML and sound fonts.

  • $kk slider (MML format songs)

New Horizons Radio

Isabelle Bot can queue and play songs from the New Horizons soundtrack.

  • $music song (Play a song by name)
  • $randommusic number (Creates a random playlist based on the number of songs requested)
  • $jump number (Jump to a specific song in the queue)
  • $queue (View the song queue)

Nintendo Switch Friend Code Sharing

Isabelle Bot can create a list of friend codes that can be fetched.

  • $friend get code (Fetches code of specific user)
  • $friend set code (Add your friend code to list)
  • $friend all (Fetches entire friend code list)
  • $friend remove name (Remove your code or someone else's as an admin)

Isabelle Bot Settings

These are server-wide settings for Isabelle Bot that only admins may use.

  • $region North/South (Sets region for NH Lookup)
  • $volume (Sets volume for Isabelle Bot)

A Project of Passion

Isabelle Bot was built from the ground up using Python 3.6.8 to coincide with HenryIsHuman's Animalese code. From there, our ideas kept growing into what would become the All-in-One Animal Crossing bot!

  • Isabelle Bot was our introduction to the Python language. We jumped in head first and have not looked back.
  • This project was made for fun by a group of friends. As a result Isabelle Bot is 100% free and has no premiums! However, you can donate to the project and buy us a cup of coffee or help us out!
